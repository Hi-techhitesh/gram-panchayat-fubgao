## ğŸš€ QUICK START (5 MINUTES)

### Step 1: Extract Files
```bash
unzip gram_panchayat.zip
cd gram_panchayat
```

### Step 2: Create Virtual Environment
```bash
# On Windows
python -m venv venv
venv\Scripts\activate

# On macOS/Linux
python3 -m venv venv
source venv/bin/activate
```

### Step 3: Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 4: Configure Environment
```bash
# Copy example file
cp .env.example .env

# Edit .env file with your settings (use any text editor)
# At minimum, change:
# - SECRET_KEY: Generate from https://djecrety.ir/
# - DEBUG: True (for development)
```

### Step 5: Setup Database
```bash
python manage.py migrate
python manage.py createsuperuser
python manage.py collectstatic --noinput
```

### Step 6: Run Server
```bash
python manage.py runserver
```

**Visit:** http://127.0.0.1:8000

**Admin Panel:** http://127.0.0.1:8000/admin/

## ğŸ“ TROUBLESHOOTING

### Issue: "No such table" error
```bash
python manage.py migrate
```

### Issue: Static files not loading
```bash
python manage.py collectstatic
```

### Issue: Admin panel not appearing
- Ensure you created a superuser: `python manage.py createsuperuser`

### Issue: Photos not displaying
- Ensure media folder exists: `mkdir media/{village,members,gallery}`
- In development, settings.py includes media URL serving

### Issue: .env file not being read
- Install python-decouple: `pip install python-decouple`
- Ensure .env file is in same directory as manage.py

## ğŸ› ï¸ CUSTOMIZATION

### Change Colors
Edit `/static/css/style.css` and update CSS variables:
```css
:root {
    --primary-color: #your-color;
    --secondary-color: #your-color;
    /* ... other colors ... */
}
```

### Add More Features
1. Create new model in `panchayat/models.py`
2. Create serializer in `panchayat/serializers.py`
3. Create viewset in `panchayat/views.py`
4. Register in `panchayat/urls.py`
5. Create template in `templates/`
6. Register in Django admin in `panchayat/admin.py`

### Change Email Settings
Edit `.env` file:
```
EMAIL_HOST=smtp.gmail.com
EMAIL_HOST_USER=your-email@gmail.com
EMAIL_HOST_PASSWORD=your-app-password
```
