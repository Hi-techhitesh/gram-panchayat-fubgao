## ğŸ”§ CUSTOMIZATION GUIDE

### Change Village Name
Edit settings and templates:
```
1. gram_panchayat/settings.py - ALLOWED_HOSTS
2. templates/base.html - Site title
3. static/css/style.css - Brand colors
```

### Add New Features
```
1. Create model in panchayat/models.py
2. Create serializer in panchayat/serializers.py
3. Create viewset in panchayat/views.py
4. Register in panchayat/urls.py
5. Create template in templates/
6. Add to admin.py
7. Create migration: python manage.py makemigrations
```

### Change Design
```
- Color variables in static/css/style.css
- Font families - Update body font-family
- Layout - Grid/Flex values
- Spacing - CSS custom properties
- Images - Replace in media/ folders
```
